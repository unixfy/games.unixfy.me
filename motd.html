About This Server

Some quick information about this gameserver
  • This server is run by Unixfy.
  • We are using ULX administration utilities.
  • Default rank is user and has no access to most commands.
  • To show commands, open up the ULX GUI with !xgui
  • For more information, scroll down to the More Information panel.
This server is private, so rules will be more relaxed than on public ones. However, rules still need to be followed!

Rules
